# vpmframe-email

Starter framework for email design; leverages [Zurb Inky](https://github.com/zurb/inky) and [Twig](https://github.com/twigjs/twig.js) for easier templating.

## Usage

1. `git clone [repo]`
2. `rm -rf .git && git init`
3. `yarn install`
4. `yarn run gulp watch`
